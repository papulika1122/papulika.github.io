Learn Where to Get Started!

HTML & CSS

ウェブページの作製に使用される言語です。HTMLとCSSを組み合わせることで、静的なページを作り上げることができます。

iQuery

素敵な動きを手軽に実装できるJavaScriptライブラリです。アニメーション効果をつけたり、Ajaxを使って外部ファイルを読み込んだりといろいろできます。

Ruby

オープンソースの動的なプログラミング言語で、シンプルさと高い生産性を備えています。大きなWebアプリケーションから小さ日用ツールまで、様々なソフトウェアを作ることができます。

PHP

HTMLだけではページの内容を変えることはできません。PHPはHTMLにプログラムを埋め込みそれを可能にします。
